# ApiClimaTempo

Api utilizando restsharp para consulta e retorno de clima

-- Projeto Asp.Net Web API

Obs. Ao criar o projeto será nescessario criar a pasta Constants e nela informar dois arquivos sendo eles para criação da ConectionString e outro para informar a key utilizada da api fornecida pela ClimaTempo (Esses arquivos e pastas podem ser modificados no codigo).
